# Debugging 
## What Went Wrong: Trouble Shooting JavaScript
* Syntax Errors: Spelling errors.
* Logic Errors: Synax is correct, but code is not doing inteded purpose. There typically isn't an error message, so they can be more difficult to find.  
* Common Errors:   
    * Is not a function: often means there is a misspelling. Pay attention to case sensitivity.  
    * Cannot access properties: may be issues with strings/variables  
    * Missing `;` before statements 
    * Missing `)` after argument list  
    * Missing `:` after property IDs
    * Missing `}` at end

# Browser Dev Tools
* Opening Dev Tools: `ctrl + shift + i OR f12` or on chrome `more tools > dev tools` or `right click > inspect` 
* DOM Inspector: Options for deleting, editinbg as HTML, element toggle states, copy/paste  
* CSS Editor: Shows rules applieed to elements, box models, editing  
* JavaScript Debugger: Sources tab in Dev tools. Includes file list, source code.  
    * Watch Expressions: Shows list items variables have been added  
    * Breakpoints: Breakpoints set on page  
    * Call Stack: Code executed to get to current line  
    * Scope: What values are visble from various parts of your code.  
* JS Console: Helpful when not running ad expected. Shows errors when running code.
