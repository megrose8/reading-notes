# Class 2: The Coder's Computer
*Key Questions*
1. What are four important features to look for in a text editor?
2. What do the following commands do? `pwd` | `ls` |
`cd` | `mkdir` | `touch`
3. Can you explain what is happening in the following scenario if these commands and arguments are entered into the command line? (Arguments are extra instructions given to a command.) | `cd projects` |
`mkdir new-project` | `touch new-project/newfile.md` | `cd ..` | `ls projects/new-project`

## Readings Notes:
+ The Command line or terminal is a text based interface to the system.
    + `PWD` Locates your current location
    + `LS` Lists the files in a given location
    + `CD` Moves you to the specified location
    + `MKDIR` Creates a new directory
    + `touch` Creates a new file
+ Key Components in a text editor: 
    + Variety of themes
    + Code completion
    + Highlighting
    + Variety of extensions

*Example scenarios:* 
+ `cd projects` brings you to the location "projects"
+ `mkdir new-project` will create a new repository named "new-project"
+ `touch new-project/newfile.md` will create a markdown file named "newfile" and place it in the `new-project` directory
+ `cd...` will bring you to the location specified
+ `ls projects/new-project` will list the files in the "new project" folder within the "project" directory
